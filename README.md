# RainRTX
C# Ray tracing renderer

In this project i'm going to develop rtx renderer running on CPU.

### Current features:
 * Sphere rendering 
 * Ground plane rendering
 * SkyBox rendering and cube mapping
 * Reflections
 * Diffuse lighting
 * Specular reflection
 * Anti-Aliasing
